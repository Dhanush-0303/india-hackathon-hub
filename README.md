# 🚀 India Hackathon Hub

An installable Progressive Web Application (PWA) built to help developers and students discover live hackathons across India.

## ✨ Features
- **User Authentication:** Secure sign-up and login using Firebase Auth.
- **Dynamic Database:** Real-time data storage and fetching via Cloud Firestore.
- **Admin Panel:** Specialized dashboard to update listings and instantly broadcast notifications.
- **Search & Filter:** Easily filter hackathons by presentation mode (Online vs. Offline).
- **Mobile Optimized:** Full PWA setup allowing direct installation on mobile homescreens or compilation via Android Studio WebView.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Modern JavaScript (ES6 Modules)
- **Backend-as-a-Service:** Firebase (Auth & Firestore)
- **Mobile Integration:** Android Studio WebView Wrapper
